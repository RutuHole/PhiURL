# PhiURL
Graph based Phishing website detection using Loopy Belief Propogation
<br>
The Phishing Website Detection System is a machine learning-based cybersecurity project designed to identify phishing websites by analyzing URL characteristics and website visual features. 
<br>The system combines URL-based feature extraction, Graphical Probability Analysis, and Loopy Belief Propogation(LBP) feature extraction to improve phishing detection accuracy. 
<br>
Users can enter a website URL, and the system predicts whether it is Legitimate or Phishing.
